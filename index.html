Welcome to RenewGrass!

The objective of RenewGrass is to allow modeling, implementation and execution of image processing workflows using Reference nets. The main functionality of RenewGrass is the invocation of Grass GIS commands from Petri nets transitions. This work is based on the JGrass project.
